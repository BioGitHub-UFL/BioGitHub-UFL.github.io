Time and place TBA: Getting started with git, github, and github pages

Need a crash course on github? Wondering how using github can make your science better? Biogithub will be putting on an introductory course on using git and github, which can be a great way to increase the transparency and visibility of your research. Then we'll talk about hosting your own website through a your github repository with github pages. Many websites (including this one) are hosted through github for free; we'll talk about the basics to get you started!
